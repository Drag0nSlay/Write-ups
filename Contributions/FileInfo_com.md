# Contribution: Identification of `.NETXML` / `.KISMET.NETXML` File Extension

**Contributor:** Aman Kothari  
**Platform:** FileInfo.com  
**Domain:** Cybersecurity
**Status:** Accepted for Review (Pending Publication)

---

## Overview

While working on wireless reconnaissance labs on the **TryHackMe** platform, I encountered a file with the extension:
`.kismet.netxml`

At the time of discovery, no authoritative public documentation existed for the `.NETXML` file extension on FileInfo.com. I initiated contact with their editorial team to help identify and document the file type.

---

## Discovery Context

- Platform: **TryHackMe**
- Lab Type: Wireless Reconnaissance / Network Analysis
- Tool Involved: **Kismet**
- File Source: Downloadable task artifacts provided during lab exercises

The file appeared as part of captured wireless network logs used for analysis.

---

## Technical Details

### File Extension
`.NETXML / .KISMET.NETXML`

### File Format
- XML-based structured data
- Human-readable
- Suitable for parsing using scripts or visualization tools

### Primary Usage

The file is generated by **Kismet**, a well-known wireless monitoring and intrusion detection tool.

It typically contains:
- SSID (network names)
- BSSID (MAC addresses)
- Signal strength
- Encryption and authentication details
- Wireless reconnaissance metadata

---

## Associated Tools

- **Kismet** – Generates `.kismet.netxml` logs
- **Wireshark** – Can assist in packet-level analysis
- **Python** – XML parsing for automated analysis
- **Kismet Log Viewer** – Native visualization
